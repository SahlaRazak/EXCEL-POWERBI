# COURSEERA COURSE ANALYSIS 2023

## Objective
   To explore and analyze Coursera course data to identify trends in ratings, difficulty, and popularity, enabling actionable insights for learners and educators."
## Dataset
   - [Source: Kaggle (original link provided in `dataset_link.txt`)](https://www.kaggle.com/datasets/tianyimasf/coursera-course-dataset)

## Tools Used
   - Microsoft Excel – Data cleaning and preprocessing
   - Power Query – Data transformation
   - Power BI – Data modeling, DAX measures, and dashboard creation

## Data Preparation
   - Removed null values and duplicates
   - Standardized sentiment labels
   - Created calculated columns and measures using DAX
